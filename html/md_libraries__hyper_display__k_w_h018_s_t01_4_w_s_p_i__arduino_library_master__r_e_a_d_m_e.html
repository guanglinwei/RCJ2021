HyperDisplay KWH018ST01 4WSPI Arduino Library

This is an extension for the HyperDisplay library that is fully configured for the KWH018ST01 module used on several SparkFun products.

Dependencies

You will also need to install these libraries, if you haven't already. Use the Arduino Library Manager or install the ZIP directly.

  • HyperDisplay Graphics Library
  • HyperDisplay ILI9163C Driver

Repository Contents

  • **/examples** - Example sketches for the library (.ino). Run these from the Arduino IDE.
  • **/src** - Source files for the library (.cpp, .h).
  • keywords.txt - Keywords from this library that will be highlighted in the Arduino IDE.
  • library.properties - General library properties for the Arduino package manager.
